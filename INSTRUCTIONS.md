# Instructions for Real-Time Multiplayer Quiz Game

This guide provides instructions on how to play the Real-Time Multiplayer Quiz Game.

## What It Does

This is a live, competitive quiz game where you can challenge your friends in real-time. All players answer questions simultaneously, and scores are updated instantly on a live leaderboard. The game also features a real-time chat, player avatars, and sound effects.

## How to Play

The game is a web application, accessible at **[https://quiz.curak.xyz](https://quiz.curak.xyz)**.

### 1. Creating a New Game (for the Host)

1.  On the home screen, enter your name and choose an avatar.
2.  Click **"Create New Game"**.
3.  You can optionally configure the game by choosing a specific question category or difficulty level.
4.  Click **"Confirm & Create"**.
5.  The game will generate a unique **Game ID**. Share this ID with your friends so they can join.
6.  You will see players appear in the lobby as they join. You can chat with them while you wait.
7.  Once everyone is in, click **"Start Game"**.

### 2. Joining an Existing Game

1.  On the home screen, enter your name and choose an avatar.
2.  Click **"Join Game"**.
3.  Enter the **Game ID** that the host shared with you.
4.  You will be taken to the game lobby. Wait for the host to start the game.

### 3. Gameplay

*   **Answer Quickly:** Read the question and select your answer as fast as you can. The faster you answer correctly, the more points you get!
*   **Live Leaderboard:** Watch the leaderboard on the side of the screen to see your rank change in real-time.
*   **Chat:** Use the in-game chat to talk with other players.
*   **Timer:** You have 15 seconds for each question. The timer bar will change color to warn you when time is running out.

### 4. After the Game

Once all questions have been asked, a final leaderboard will be displayed showing the winner. Your personal statistics (like win rate and accuracy) will be automatically updated and saved for you to view.

Click **"Play Again"** to return to the lobby for another round!
